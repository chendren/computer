# LCARS Design Reference

- Orange/Amber: #FF9900 (primary text, borders, buttons)
- Peach: #FF9966 (secondary elements)
- Lavender: #CC99CC (headers, labels)
- Blue: #9999FF (tertiary accents)
- Light Blue: #99CCFF (data text, links)
- Rose: #CC6699 (alerts, accents)
- Gold: #FFCC00 (highlights, warnings)
- Background: #000000
- Panel BG: #0a0a1a
- Dim text: #996600
- Grid: #333333
